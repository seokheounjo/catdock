주요 기능

🏢

계층형 팀 구조

Director → Leader → Member 조직도. 자동 위임, 상향 보고, 체인 보고로 실제 회사처럼 운영됩니다.

💰

예산 관리

에이전트별 월 예산 한도 설정. 80% 경고, 100% 자동 중지. API 비용을 완벽하게 통제합니다.

✅

승인 게이트

위임이나 에이전트 생성 전 사용자 승인을 요청합니다. 120초 타임아웃으로 자동 거부.

💬

1:1 채팅 & 그룹 채팅

에이전트와 개별 대화하거나, 여러 에이전트가 순서대로 토론하는 그룹 채팅을 지원합니다.

🛡

오류 자동 복구

Member → Leader → Director 다단계 에스컬레이션. 워치독이 멈춘 프로세스를 자동 감지하고 복구합니다.

🔌

MCP 통합

Global, Team, Agent 3단계 MCP 설정. 헬스체크, 자동 재시도, 장애 보고까지 지원합니다.

🎮

커맨드 센터

모든 에이전트를 한 화면에서 모니터링. 계층별 채팅, 실시간 상태, 일괄 명령을 지원합니다.

📊

활동 트레이싱

모든 도구 호출, 위임, 보고, 예산 이벤트를 실시간으로 기록하고 대시보드에서 확인합니다.

🎨

Figma 연동

Figma MCP 서버로 디자인 파일을 참조하여 React + Tailwind 코드로 변환합니다.

설치 방법

3분이면 시작할 수 있습니다.

사전 준비 (필수)
아래 2개를 먼저 설치해주세요:
1. Node.js 18+ 다운로드 & 설치
2. Claude Code CLI: 터미널에서 npm install -g @anthropic-ai/claude-code 실행 후 claude로 로그인

Windows 설치 파일 권장

  1. 위의 "Windows 다운로드" 버튼 클릭 (94MB)
  2. virtual-company-1.7.0-setup.exe 실행
  3. 설치 경로 선택 → 설치 완료
  4. 바탕화면의 "Virtual Company" 아이콘 더블클릭
  5. 셋업 마법사: CLI 확인 → 작업 디렉토리 선택 → 완료

소스에서 실행 개발자

  1. git clone https://github.com/seokheounjo/catdock.git
  2. cd catdock
  3. npm install -g pnpm (없다면)
  4. pnpm install
  5. pnpm dev

왜 이 도구들이 필요한가요?

AI 코딩 에이전트 시대, 혼자서는 감당할 수 없는 복잡성을 관리합니다.

🤖

에이전트 폭주 방지

예산 한도 + 승인 게이트로 AI 에이전트의 무분별한 API 호출과 자동 위임을 통제합니다.

🏢

팀처럼 운영

Director가 작업을 분배하고, Leader가 팀원을 관리하고, Member가 실행합니다. 실제 회사 구조.

💻

멀티 에이전트 터미널

Claude Code, Codex, Gemini CLI를 각각 다른 워크스페이스에서 동시 실행. 알림으로 상태 파악.

💰

비용 투명성

에이전트별 API 비용 추적, 월별 자동 리셋, 80% 경고, 100% 자동 중지. 깜짝 청구서 없음.

🔄

세션 복원

wumx는 터미널 화면까지 복원. CatDock은 대화 히스토리와 위임 상태를 유지합니다.

🛡

자동 복구

에이전트 오류 시 상위자에게 보고, 백업 디렉터로 자동 전환. 사람 개입 없이 복구.

제품 비교

두 제품은 서로 다른 문제를 해결합니다. 함께 사용하면 최강.

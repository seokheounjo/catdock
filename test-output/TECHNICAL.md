#### 사용된 Node.js API
- **`console.log()`**: 표준 출력 스트림으로 텍스트 출력

- **Node.js**: v14.0.0 이상 권장 (console.log는 모든 버전 지원)

#### 성능 특성

- **메모리 사용**: < 20MB (Node.js 런타임 포함)
- **파일 크기**: 109 bytes (UTF-8 인코딩)

## 문제 해결

### 일반적인 오류
1. **"node: command not found"**
   - 해결: Node.js 설치 확인 (`node --version`)

2. **"Cannot find module"**
   - 해결: 현재 디렉토리에 hello.js 파일 확인

3. **권한 오류**
   - 해결: 파일 실행 권한 확인 (`chmod +x hello.js`)

### 디버깅 방법
```bash
# Node.js 디버그 모드
node --inspect hello.js

# 상세 로그 출력
NODE_DEBUG=* node hello.js
```
